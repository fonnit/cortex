## Decisions Made

1. **Schema simplification**: Dropped `role`, `company`, `relationship`. Replaced with `type` (free-text). All historical semantics ("owner", "known_person + relationship") are captured via free-text type values.
2. **type free-text with datalist**: No enum constraint. Datalist seeds suggestions from existing types + "owner" and "company" defaults.
3. **contextBlock format**: `"Daniel Fonnegra (owner), Acme Corp (company), Maria (partner)"` — no email in prompt, name+type only.
4. **Identity suggest placement**: After the `cx-axes` block, before `cx-card-actions`. No modal, no toast. Inline dismiss via Skip.
5. **identities prop optional**: `identities` defaults to `[]` — backward compatible, TriageView threads it down.
