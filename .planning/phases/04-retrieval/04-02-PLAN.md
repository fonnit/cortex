<!-- ANN query pattern using @neondatabase/serverless -->
```typescript
import { neon } from '@neondatabase/serverless'
const sql = neon(process.env.DATABASE_URL!)
// Inner product distance (<#>) with halfvec — returns most similar first
const rows = await sql`
  SELECT id, filename, axis_type, axis_from, axis_context,
         confirmed_drive_path, proposed_drive_path, ingested_at,
         source_metadata,
         (embedding <#> ${`[${queryVec.join(',')}]`}::halfvec) AS score
  FROM "Item"
  WHERE status = 'filed' AND embedding IS NOT NULL
  ORDER BY embedding <#> ${`[${queryVec.join(',')}]`}::halfvec
  LIMIT 20
`
```
Note: `<#>` is inner product distance — lower = more similar for normalized vectors (OpenAI outputs are L2-normalized). ORDER BY ASC gives most similar first.

**Langfuse trace:** open trace before any LLM/DB work:
```typescript
import Langfuse from 'langfuse'
const lf = new Langfuse()
const trace = lf.trace({ name: 'ask', userId, input: { question: body.question } })
```

**Embed query** (timed for latency):
```typescript
const t0 = Date.now()
const embedSpan = trace.span({ name: 'embed-query', input: { text: body.question } })
const [[queryVec]] = await embedTexts([body.question])  // 512d
embedSpan.end({ output: { dims: queryVec.length } })
```

**ANN retrieval** (raw SQL — Prisma cannot handle halfvec operator):
```typescript
const retrieveSpan = trace.span({ name: 'pgvector-ann', input: { limit: 20 } })
const sql = neon(process.env.DATABASE_URL!)
const vecStr = `[${queryVec.join(',')}]`
const rows = await sql`
  SELECT id, filename, axis_type, axis_from, axis_context,
         confirmed_drive_path, proposed_drive_path, ingested_at,
         source_metadata
  FROM "Item"
  WHERE user_id = ${userId}
    AND status = 'filed'
    AND embedding IS NOT NULL
  ORDER BY embedding <#> ${vecStr}::halfvec
  LIMIT 20
`
retrieveSpan.end({ output: { count: rows.length } })
```

**Build synthesis prompt:** Take top 5 rows (or fewer if <5) as citations. Format each as:
```
[1] filename: {filename} | path: {drive_path} | type: {axis_type} | filed: {ingested_at_date}
```
where `drive_path = confirmed_drive_path ?? proposed_drive_path ?? 'unknown'`.

System prompt:
```
You are Cortex, a personal AI assistant. Answer the question using ONLY the provided documents.
Cite sources inline as [1], [2], etc. Use only citation numbers that correspond to listed documents.
Limit to ≤5 distinct citations. Be concise. Output ONLY the answer text with inline [N] citations — no preamble.
```

**Haiku synthesis:**
```typescript
const synthSpan = trace.span({ name: 'haiku-synthesis', input: { docs: top5.length } })
const msg = await anthropic.messages.create({
  model: 'claude-haiku-4-5',
  max_tokens: 800,
  system: systemPrompt,
```

**Parse answer into paragraphs + cites:** Split rawAnswer on double newlines. For each paragraph, extract `[N]` patterns to build `cites: number[]` array. Strip the inline `[N]` markers from the text (keep as anchor links in UI). Return the structured `AskResponse`:

```typescript
const answer = paragraphs.map(p => ({
  text: p.replace(/\[(\d+)\]/g, '').trim(),
  cites: [...p.matchAll(/\[(\d+)\]/g)].map(m => Number(m[1])),
}))
const sources = top5.map((row, i) => ({
  n: i + 1,
  title: (row.filename as string) ?? (row.source_metadata as any)?.subject ?? 'untitled',

  when: new Date(row.ingested_at as string).toLocaleDateString('en-US', { month: 'short', day: 'numeric', year: 'numeric' }),
}))
const latencyMs = Date.now() - t0
```

**Langfuse close:**
