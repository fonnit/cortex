## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| consumer process → `claude` CLI subprocess | Untrusted file-system content (filenames, paths) crosses into argv |
| consumer process → Vercel API | Outbound HTTP carrying classification verdicts; bearer-token auth |
| Vercel API → consumer (response body) | Untrusted JSON from QueueResponse / TaxonomyInternalResponse parsed by Zod before use |
| `claude` CLI stdout → consumer | Untrusted text — must be JSON-extracted via regex + JSON.parse + Zod, never `eval`/Function |
| internet → `app/api/taxonomy/internal` | Public route guarded ONLY by CORTEX_API_KEY shared secret |
